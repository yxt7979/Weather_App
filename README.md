# my_first_app

**learn from [视频地址](https://www.youtube.com/watch?v=JLc-hWsPTUY)**

> A Vue.js project

##  Setup Vue

```bash
npm install --global vue-cli
```

## New Project
```bash
vue init webpack my_first_app
```

## Run
```bash
cd my_first_app
npm run dev
```
